Latest Projects

Underwater surveillance robot

An underwater robot powered by Pixhawk and controlled via QGroundControl software offers a solution for aquatic exploration and operations. This wired ROV (Remotely Operated Vehicle) provides reliable communication and power transfer, ensuring uninterrupted operations even in challenging underwater environments. Equipped with Pixhawk flight controller, it performs tasks such as search & rescue operation, live video streaming etc.

Hull cleaning robot

A hull cleaning robot is an innovative solution designed to efficiently clean the underwater surfaces of ship hulls and other marine structures. Equipped with magnetic wheels or adhesion technology, it securely attaches to the hull, ensuring stability during operation. The robot uses advanced cleaning brushes, water jets, or scrubbing mechanisms to remove biofouling, algae, and debris, improving the vessel's performance and fuel efficiency.

Latest Projects

Line Follower robot

A line follower robot is a simple yet effective autonomous robot designed to follow a predefined path, usually a black or white line on a contrasting surface. It uses infrared (IR) sensors to detect the line's position and employs a control system to adjust its motors, ensuring accurate tracking.

Obstacle avoidance robot

n obstacle avoidance robot designed using ROS (Robot Operating System), Gazebo simulation, and Arduino software. The robot utilizes ROS for real-time data processing and communication, enabling integration of sensors like ultrasonic for obstacle detection. Gazebo simulation allows for testing and optimizing the robot's behavior in a virtual environment.
